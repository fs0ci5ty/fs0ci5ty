## About

Security researcher focused on offensive tooling, exploit development, and low-level vulnerability research. This account serves as a public archive of tools, scripts, and technical write-ups produced during research, CTF competitions, and red team engagements.

All published material is intended for authorized use, academic study, and the advancement of the security community.

---

## Research Interests

- **Exploit Development** — Binary exploitation, shellcoding, ROP chain construction, heap feng shui
- **Vulnerability Research** — CVE analysis, fuzzing, static and dynamic reverse engineering
- **Red Team Tooling** — Custom implants, C2 infrastructure, AV/EDR evasion techniques
- **Web Security** — OWASP attack surface, deserialization, SSRF, logic flaws
- **Network Offense** — Packet manipulation, protocol abuse, man-in-the-middle frameworks
- **OSINT & Recon** — Automated reconnaissance, passive fingerprinting, attack surface mapping
